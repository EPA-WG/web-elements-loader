DETAILS/SUMMARY for menu

Fully works only in Chrome and Firefox. In Edge there is no collapsible behavior. Check level up for advised solution.

The triangle which reflects the open state could be hidden but there is no standard way. In this page Chrome and Firefox code co-exist: 

header details>summary { list-style: none;padding: 0.5rem; }
header details>summary::-webkit-details-marker {
    display: none;
}

Note: Unfortunately, at this time there's no built-in way to animate the transition between open and closed.
