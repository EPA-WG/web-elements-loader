Checkbox as menu trigger

Menu could be triggered by checkbox in CSS, the checkbox UI could be replaced with icon image.

This is a simplest method of menu trigger after DETAILS TAG use. It is focus-able and keyboard navigation capable.

Unfortunately Edge browser does not support the checkbox UI overriding. Solution is to hover it by LABEL with icon UI and non-native styling on checkbox focus. The label would simulate checkbox appearance, while behavior would be defined by checkbox.
